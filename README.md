To run,

### Create a Virtual Environment

```bash
python3 -m venv env
```

### Activate the Virtual Environment

#### For MacOS

```bash
source ./env/bin/activate
```

#### For Windows

```bash
.\env\Scripts\activate
```

### Install the requirements

```bash
pip install -r requirements.txt
```

### Run

```bash
python3 src/main.py
```
